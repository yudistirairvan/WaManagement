Aplikasi Dashboard WA UMKM berbasis React & Gemini AI untuk manajemen pesan otomatis. Fitur utama: Blast WhatsApp massal, pengelolaan chat real-time, dan Chatbot AI yang terintegrasi Knowledge Base bisnis. Memungkinkan UMKM merespons pelanggan 24/7 secara cerdas, personal, dan efisien menggunakan model Gemini-3-flash-preview.
